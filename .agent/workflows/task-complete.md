---
description: Görev tamamlandığında otomatik commit atılması için izlenecek adımlar
---
1. Tüm değişikliklerin kaydedildiğinden emin ol.
2. `git add .` komutunu çalıştır.
3. `git commit -m "<tip>(<kapsam>): <açıklama>"` formatında, yapılan değişiklikleri özetleyen bir commit mesajı ile commit at.
4. Commit mesajının proje standartlarına (Türkçe, küçük harf, nokta yok) uygun olduğundan emin ol.
